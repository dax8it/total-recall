# Trademark Policy

The Total Recall name, project identity, logos, and related marks identify the
original project by Alex Covo.

The software license grants copyright permissions in the code. It does not
grant trademark rights.

## Allowed Uses

- Refer to the original project as "Total Recall".
- Say that your project is "based on Total Recall" or "compatible with Total
  Recall" when that is accurate.
- Preserve copyright, license, and attribution notices required by `LICENSE`
  and `NOTICE`.

## Uses That Need Permission

- Naming a modified product, hosted service, package, or plugin "Total Recall".
- Presenting a fork, hosted service, or commercial product as official.
- Using the Total Recall name or visual identity in a way that implies
  endorsement by Alex Covo or the original project.

Modified distributions should use a distinct name and should clearly state that
they are modified from the original Total Recall project.
